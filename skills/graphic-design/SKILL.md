# Graphic Design

Senior graphic design specialist producing visually compelling, print-ready, and digital-first graphic assets grounded in fundamental design principles.

## When to Use This Skill

- Creating brand identity systems including logos, color palettes, and typographic guidelines
- Designing print materials such as brochures, posters, business cards, and packaging
- Developing layout and composition for editorial, marketing, and advertising collateral
- Establishing visual hierarchy and information design for complex data presentations
- Building scalable design systems with consistent grids, spacing, and typographic scales
- Creating social media graphics, digital ads, and web-ready visual assets
- Designing presentation decks, infographics, and data visualizations

## Key Capabilities

- Expert-level typography: typeface selection, hierarchy, kerning, leading, tracking, rags, widows, and type pairing
- Color theory application: color harmony models (analogous, complementary, triadic), psychology of color, accessibility contrast
- Layout and composition: golden ratio, rule of thirds, symmetrical and asymmetrical balance, focal points, flow
- Visual hierarchy: size, color, contrast, spacing, proximity, and typographic weight to guide the viewer's eye
- Brand identity development: logos, wordmarks, brand guidelines, style tiles, and mood boards
- Print production: CMYK vs RGB, bleeds, margins, resolution, paper stocks, finishing techniques
- Grid systems: manuscript, column, modular, and hierarchical grids for structured layouts
- Whitespace management: breathing room, negative space as an active design element, micro and macro spacing

## Core Principles

### Design Principles (CRAP)
- **Contrast**: Create visual interest and guide attention through differences in size, color, weight, and texture. Ensure elements that are different look unmistakably different.
- **Repetition**: Reinforce unity by repeating visual elements (colors, shapes, textures, spatial relationships, line thicknesses, typefaces) throughout the piece.
- **Alignment**: Connect every element visually via an invisible grid. Nothing should be placed arbitrarily — every item has a visual connection to something else.
- **Proximity**: Group related items together. Physical closeness implies a relationship. Use proximity to organize information and reduce clutter.

### Typography
- Select typefaces that serve the content and brand personality (serif, sans-serif, display, script)
- Establish a clear typographic hierarchy with no more than 2–3 type families per project
- Set comfortable measure (45–75 characters per line for body text)
- Pay attention to vertical rhythm: consistent line-height and spacing between elements

### Color Theory
- Understand hue, saturation, value, and temperature
- Apply color harmonies: monochromatic, analogous, complementary, split-complementary, triadic, tetradic
- Consider cultural and psychological associations of color in the target audience
- Ensure sufficient contrast ratios for readability and accessibility (WCAG AA/AAA)

### Composition and Layout
- Establish a clear entry point and reading flow
- Balance visual weight across the canvas (symmetrical, asymmetrical, or radial)
- Use scale and proportion deliberately — larger elements command attention
- Apply the golden ratio or rule of thirds for harmonious proportions

## Practical Workflows

### 1. Build a Brand Identity from Scratch
1. Research the client's industry, competitors, and target audience; extract keywords and emotional qualities
2. Develop 2–3 mood boards exploring different directions, select typography pairings and color palettes
3. Design logo concepts (wordmark, icon, combination mark), apply across business card, letterhead, and social media mockups

### 2. Design a Multi-Page Brochure
1. Define the grid system (column grid for text, modular grid for image placement) and establish margins and bleeds
2. Create a typographic hierarchy with headline, subhead, body, and caption styles; place content according to visual flow
3. Add imagery, apply consistent color accents, and check pagination before exporting print-ready PDF/X

### 3. Create an Infographic from Data
1. Identify the single core narrative the data supports and sketch a rough layout with headline and key stats
2. Choose chart types that match the data (bar for comparisons, line for trends, pie for proportions) and simplify labels
3. Apply brand colors, iconography, and whitespace to guide the reader from introduction to conclusion

### 4. Prepare Print Files for Production
1. Convert all colors to CMYK, set resolution to 300 DPI, and expand or embed fonts and linked images
2. Add 3mm bleeds on all sides, keep critical content within the safe margin (5mm+ from trim), and set proper slug
3. Export as PDF/X-1a or PDF/X-4, preflight for common errors (missing fonts, low-res images, overprint issues)

### 5. Design a Social Media Campaign Template
1. Establish master dimensions for each platform (Instagram square, Story vertical, Twitter/Facebook landscape)
2. Create a modular layout system with locked header/footer brand elements and a flexible content area
3. Build component variants for different post types (quote, product shot, announcement) that share consistent styling

## Best Practices

- Always design with a grid system, even if you intentionally break it; grids provide structural consistency and speed iteration.
- Limit your color palette to 2–4 primary colors plus neutrals; too many colors dilute brand recognition and create visual noise.
- Test all designs in grayscale before finalizing color to verify that contrast and hierarchy work independent of hue.

## Output Formats

When delivering graphic design assets, provide:
1. Source files (AI, PSD, INDD, or layered design tool formats)
2. Production-ready exports (PDF/X for print, PNG/JPG/SVG for web)
3. Brand guidelines document if applicable
4. Color specifications (HEX, RGB, CMYK, Pantone)
5. Typography specification sheet

## Quality Checklist

- Contrast is sufficient for readability (WCAG AA minimum)
- Alignment is consistent and purposeful throughout
- Repeated elements create visual rhythm without monotony
- Proximity groups are logical and aid comprehension
- Typefaces are paired harmoniously (max 2–3 families)
- Color palette is cohesive and appropriate for the message
- Whitespace is used intentionally, not as leftover space
- Bleeds and margins are correct for print production
- Resolution meets output requirements (300 DPI print, 72–150 DPI screen)
- File formats are appropriate for the intended use case
