# First Principles Thinking

## Core Process
1. **Identify the problem/assumption**: what's the conventional wisdom?
2. **Break it down**: decompose into fundamental truths that are known to be true
3. **Rebuild up**: create new solutions from those fundamentals, ignoring existing constraints

## Decomposition Method
- What do we KNOW to be true? (not what we assume)
- What physics/ math/ logic constraints exist? (not human-made ones)
- What can be proven or disproven?
- Strip away: industry norms, "that's how it's always done", expert opinions

## Rebuilding
- Start with the fundamentals as building blocks
- Combine them in novel ways
- Don't stop at one alternative — generate multiple paths
- Test each path against reality

## Questions to Drive First Principles
- "If I were starting from zero, how would I solve this?"
- "What would this look like if I had no budget/ time/ resource constraints?"
- "What spending half the resources?"
- "What if we reversed the goal?"

## Mentals Models That Complement
- **Inversion**: solve backwards — what would guarantee failure? Avoid those.
- **Occam's Razor**: simplest explanation is usually correct
- **Reductionism**: complex phenomena from simple parts
- **Thought experiments**: test ideas mentally before acting

## Application
- Tech: "What's the fundamental math/compsci principle?"
- Business: "What does the customer actually need? What's the atomic unit of value?"
- Strategy: "If this entire industry didn't exist, what would we build?"
